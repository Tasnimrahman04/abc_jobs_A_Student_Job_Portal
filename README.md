# abc_jobs_A_Student_Job_Portal


abc_jobs is a student job portal built with PHP and MySQL. The platform connects students with companies by allowing job posting, job searching, applications, and CV management.

## Features
- Student, Company, and Admin login
- Job search and filtering
- Job posting and management
- Company management
- CV creation and editing
- Applicant tracking

## Technologies
- HTML, CSS, Bootstrap, JavaScript
- PHP
- MySQL

## Run Locally
1. Clone the repository
2. Move the folder to XAMPP htdocs
3. Start Apache and MySQL
4. Import the database file
5. Run the project in browser
